# Тест-кейсы для формы валидации email-адреса

## Требования

- допустимые символы значения email-адреса: латиница, цифры, знаки `! # $ % & ' * + — / = ? ^ _ { | } ~`
- email-адрес должен иметь валидный паттерн
- при вводе значения в инпут не должна происходить валидация
- при клике на кнопку «Проверить» должна происходить валидация email-адреса
- если email-адрес валидный, выводится alert с текстом «Прекрасный email-адрес»
- если email-адрес не валидный, выводится ошибка красного цвета над кнопкой «Проверить» с текстом «Введен некорректный email»

---

## Блок 1: Позитивные сценарии (валидные email-адреса)

### 1. Ввод валидного email-адреса со стандартным форматом

**Шаги воспроизведения:**

1. Ввести в поле ввода email «k@academy.ru»
2. Нажать кнопку «Проверить»

**Ожидаемый результат:** Появился alert с текстом «Прекрасный email-адрес»

### 14. Исправление невалидного email на валидный

**Шаги воспроизведения:**

1. Ввести в поле ввода email «kacademy.ru»
2. Нажать кнопку «Проверить»
3. Убедиться, что появилось сообщение об ошибке
4. Изменить значение в поле на «k@academy.ru»
5. Нажать кнопку «Проверить»

**Ожидаемый результат:** Сообщение об ошибке исчезло, появился alert с текстом «Прекрасный email-адрес»

---

### 15. Повторная валидация без изменения значения

**Шаги воспроизведения:**

1. Ввести в поле ввода email «k@academy.ru»
2. Нажать кнопку «Проверить»
3. Закрыть alert
4. Снова нажать кнопку «Проверить»

**Ожидаемый результат:** Снова появился alert с текстом «Прекрасный email-адрес»
